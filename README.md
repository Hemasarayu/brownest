Brownest: Smart Family Finance App

Brownest is a modern React Native application designed to help families build healthy financial habits together. It empowers parents and teens with tools to manage spending, track savings goals, monitor allowances, and develop long-term financial literacy.

* Features
->Secure Authentication

->Login & Signup flows

->Social sign-in (Google / Facebook)

->Smooth onboarding UI

* Family Dashboard

->Unified view of total family balance

->Individual account cards for parents & kids

->Breakdown of monthly savings vs. expenditure

* Quick Actions

->Transfer pocket money instantly

->Create new savings goals

->One-tap shortcuts from the Home screen

* Transactions Feed

->Real-time list of expenses, allowances, and savings

->Clean icons and category tags

->Sort & filter transactions

* Savings Goals

->Create personalized goals

->Visual trackers with progress percentage

->Celebrate milestones when goals are reached

* Parent Controls

->Approve or decline purchase requests

->Set spending limits

->Monitor teen activity & encourage responsible habits

* Insights & Analytics

->Monthly savings trends

->Spending categories breakdown

->Family financial progress overview

*Community & Testimonials

->Family success stories

->Motivation to stay consistent with savings

->Positive money culture reinforcement

*Why Brownest?

->Family-first financial literacy with shared accountability

->Transparent spending for every family member

->Engaging visuals that help teens learn better

->Simple, modern UI made for real-life family finance

*Tech Stack
->Layer	Technology
->Frontend	React Native
->Navigation	React Navigation (stack + bottom tabs)
->Auth	Firebase Auth / OAuth (Google, Facebook)
->State Management	Context API or Redux
->Styling	Styled-components + custom theme
->Backend (Optional)	Firebase Firestore or Node.js API
